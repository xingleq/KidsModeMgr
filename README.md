# Kids Mode Manager (儿童模式管理工具)

Windows 儿童电脑使用时间管理工具，限制连续使用时长，超时强制锁屏并执行休息策略。

## 功能特性

| 功能 | 说明 |
|------|------|
| 强制锁屏 | 达到时间上限后自动断开用户会话 |
| 强制休息 | 休息期内重新登录会被立即踢出 |
| 开机自启 | 安装即自动启动，无需手动干预 |
| 计时持久化 | 关机重启不会重置当日已用时间 |
| 防时间篡改 | 使用系统运行时间（非系统时钟）判断休息期，修改系统时间无效 |
| 注册表 ACL | 安装时自动锁定注册表，普通用户无法篡改配置 |
| 崩溃自恢复 | 服务被异常终止后 5 秒内自动重启 |
| 锁屏前预警 | 剩余 5 分钟和 1 分钟时在用户桌面弹窗提醒 |
| 敏感操作保护 | 停止/卸载服务需要验证管理员密码 |
| 实时状态面板 | GUI 实时显示今日已用、距锁屏剩余、休息期倒计时 |
| 使用记录日志 | 每次锁屏事件写入日志，可在 GUI 中查看 |

## 使用说明

### 1. 启动管理工具
双击 `kids_mode_mgr.exe`，程序自动申请管理员权限。

### 2. 参数设置

- **最大使用时间**：输入分钟数（例如 30），旁边实时显示换算秒数
- **强制休息时间**：输入分钟数（例如 5）
- **开启强制休息**：勾选后生效

点击 **保存配置** 后，若服务运行中，工具会询问是否立即重启服务生效，无需手动操作。

### 3. 安装服务（首次使用）
点击 **安装服务**，程序会：
1. 注册 Windows 服务并设为开机自动启动
2. 立即启动服务
3. 配置崩溃后自动恢复策略
4. 锁定注册表 ACL（普通用户只读）

### 4. 实时状态面板
打开管理工具后，界面中部每 3 秒自动刷新：
- 今日已用时间 / 上限
- 距下次锁屏剩余时间
- 休息期剩余倒计时（红色显示）

### 5. 查看使用记录
点击 **查看使用记录** 用记事本打开日志，记录每次锁屏的时间和当日累计使用时长。

## 安全机制说明

| 攻击方式 | 对应防护 |
|----------|----------|
| 修改系统时间跳过休息期 | 使用 GetTickCount64（系统运行时间）验证，不受时钟影响 |
| 用 regedit 篡改配置/休息时间 | 注册表 ACL 限制普通用户只读 |
| 任务管理器杀死服务 | 服务失败恢复策略，5 秒内自动重启 |
| 关机重启绕过计时 | 每分钟持久化已用时间到注册表，启动时恢复 |

> **注意**：如果孩子拥有管理员账户权限，以上保护均可被绕过。建议为孩子创建标准用户账户，由家长账户运行本工具进行管理。

## 注册表路径

`HKLM\SOFTWARE\KidsModeMgr`

| 键名 | 类型 | 说明 |
|------|------|------|
| MaxUsage | DWORD | 最大使用时间（秒） |
| MandatoryRest | DWORD | 强制休息时间（秒） |
| EnableRest | DWORD | 是否启用强制休息（0/1） |
| LastForceLockTime | REG_SZ | 上次锁屏 Unix 时间戳 |
| CurrentUsage | DWORD | 当前已用秒数（服务写入，GUI读取） |
| DailyUsage_YYYYMMDD | DWORD | 当日累计使用秒数（跨天自动失效） |

## 日志路径

`%ProgramData%\KidsModeMgr\usage.log`

## 常见问题

**Q: 修改时间后没生效？**
A: 保存配置后工具会自动询问是否重启服务，确认即可。

**Q: 孩子怎么绕过了限制？**
A: 检查孩子是否有管理员账户权限。建议使用标准用户账户，这是最有效的系统级隔离手段。
